\doxysubsection{Detailed Description}
Interface for sprite objects. 

This interface inherits from {\ttfamily \doxylink{classarcade_1_1IObject}{IObject}} and defines functionalities specific to sprite objects. Sprites are graphical elements typically rendered using textures.
